# TwitterSched - Api


## Prerequisite

Python versions covered:
* `Python 3.5`
* `Python 3.6`
* `Python 3.7`
* `Python 3.8`


## Getting started

### Init database

```sh
pipenv run flask init-db
```


### Run the app

```sh
pipenv run flask run --host=0.0.0.0 --port=8080
```
Browse the app on localhost port 8080.


## Environment

### Development

```sh
pipenv update --dev
```

### Production-like

```sh
pipenv update
```
